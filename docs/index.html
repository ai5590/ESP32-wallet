Обновление

Если установлено из магазина

Обновляется автоматически через браузер.

Проверить страницу обновления

Если установлено вручную

  1. Скачайте новый zip-файл со страницы выше.
  2. Распакуйте его в новую папку.
  3. Перезагрузите расширение в браузере и укажите новую папку/manifest.

Пошаговая ручная установка

Chrome / Edge / Opera

  1. Откройте страницу расширений браузера.
  2. Включите режим разработчика.
  3. Нажмите Load unpacked.
  4. Выберите папку распакованной сборки Chromium.

Firefox

  1. Откройте about:debugging#/runtime/this-firefox.
  2. Нажмите Load Temporary Add-on....
  3. Выберите manifest.json из папки сборки Firefox.

О проекте

ESP32 Wallet — минималистичный Solana-кошелёк как браузерное расширение, совместимый с Wallet Standard и legacy window.solana. В текущей версии подпись выполняется локально в расширении.

Следующий этап: вынести хранение ключей и подпись на внешнее ESP32-устройство через отдельный transport-слой.

FAQ

Можно ли установить/обновить расширение полностью автоматически прямо с сайта?

Полностью бесшовная автоустановка запрещена политиками браузеров. Надёжный путь для пользователей — установка через официальные магазины расширений.

Почему на сайте есть ручные zip-сборки?

Это запасной путь до публикации в сторе и для тестирования. Сайт отдаёт готовые сборки, чтобы пользователь мог установить расширение вручную.
